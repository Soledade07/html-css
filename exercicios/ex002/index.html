Parágrafos e quebras de linha

eu posso escrever um Parágrafos com quantas linhas eu quiser e de qualquer jeito apenas ele(o paragrafo) tem que estar entre as tags p e /p (tudo é colocado na mesma linha o enter não é interpretado)

se eu quiser que apareça na explicação acima, a representação literal das tags <p> e </p>, devo usar caracteres especiais:
lt=less then (menor que) gt=bigger then (maior que)

para quebrar as linhas deve ser usado a tag br ou <br> ex: quebra de linha só com enter não faz. quebra de linha
com a tag br faz.

alguns simbolos especiais Δ € $ ® ™ £

alguns emojji: para isso vamos acessar o site emojipedia.org este site contem uma cotanea de praticamente todos os imoji utilizados. no site você pode copiar o emoji propriamente dito ou sua representação exa-decimal pq essa representação funciona para todos os sites, já você colar o emoji direto no código pode ser que não funcione. dessa forma vc deve colocar a representação exa-decimal 🖖 🤓
